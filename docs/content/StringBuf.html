class StringBuf
A String buffer is an efficient way to build a big string by appending small elements together.
function new() : Void
Creates a new string buffer.
function add(?x : Dynamic) : Void
Adds the representation of any value to the string buffer.
function addChar(c : Int) : Void
Adds a character to the string buffer.
function addSub(s : String, pos : Int, ?len : Int) : Void
Adds a part of a string to the string buffer.
function toString() : String
Returns the content of the string buffer. The buffer is not emptied by this operation.
